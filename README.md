To run deployed project: https://eliftech-test-task-iota.vercel.app

To run locally:
1. git clone https://github.com/pvitalii/eliftech-test-task
2. cd eliftech-test-task/backend
3. npm install
4. npm run build
5. npm run test

In another terminal:

6. cd eliftech-test-task/frontend
7. npm install
8. npm run start
